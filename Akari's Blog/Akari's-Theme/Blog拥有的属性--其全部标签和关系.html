Blog拥有的属性--其全部标签和关系

新增功能后，你需要把新的标签和关系配置到 Akari's Blog页面拥有的属性里！由于属性和标签太多了，不方便修改编辑，所以有了这个辅助页面。这些属性其实就等于是blog的配置config。

配置

配置前先把 Akari's-Theme笔记的下方两个属性去掉，否则会导致关系配置时搜到不配置文件。等配置好后再把这两个属性加回去。

#shareHiddenFromTree(可继承的) 

#archived(可继承的) 

  • #shareHiddenFromTree
    • 范围：共享子树（~share）
    • 作用：把这条笔记从共享站点的「左侧目录树」里隐藏掉，但笔记本身仍然可以通过 URL 直接访问。
  • #archived
    • 范围：任意笔记
    • 作用：把笔记标记为“已归档”。
    • 视觉上：标题会变灰、出现小归档图标；
    • 功能上：
      – 默认不再出现在“全文搜索”结果里（除非你在搜索框里显式勾选 Include archived）；
      – 不再出现在“快速搜索”（Ctrl+P）的自动补全里；
      – 保护树：运行“删除已完成分支”等维护脚本时会被跳过。

注释

1. 站点基础信息 

# ======================= 1. 站点基础信息 ========================
# 博客标题、作者、SEO、备案等「一眼能看到的文字」

2. 视觉 / 风格开关

# ======================= 2. 视觉 / 风格开关 =====================
# 背景图、头像、图标等「长什么样」由它们决定

3. 页面功能总开关

# ======================= 3. 页面功能总开关 ======================
# 一句话就能开/关的「大功能」

                                

4. 社交图标链接

# ======================= 4. 社交图标链接 ========================
# 侧边栏一排小图标，顺序即数组顺序；留空则不显示

 

5. 前端资源（relation 指向文件笔记）

# ======================= 5. 前端资源（relation 指向文件笔记）=====
# 换主题色、换代码高亮风格、升级组件库 → 换这些文件即可

                          

6. 模板与结构控制

# ======================= 6. 模板与结构控制 ========================
# 决定「用什么模板渲染」+「子笔记怎么排序」

                                   

7. 杂项 / 预留钩子

# ======================= 7. 杂项 / 预留钩子 =========================
# 角色卡片下方跳转第三方的超链接，不想显示自己不常用的，就删除对应的

8. 其他注意事项

  • 在主题根目录拥有的属性中添加#swapInfoCard - 可以交换主页左右边栏（博主信息与说说）的位置。目前修改后的布局会导致文章大纲导航侧边栏与笔记目录和信息卡片在左侧重叠，不建议启用

9.z-index元素堆叠顺序

  • shmaur-mainCss
    • .code-toolbar  代码块任务栏 红绿灯 栏样式  z-index: 1;
    • #toc-pane  文章右侧页面大纲，  z-index: 1;
    • #catalogue,#outline 手机端目录和大纲按钮样式 z-index: 2;
    • #onGoTop 回到顶部按钮   z-index: 2;
    • .searchContainerStyle  搜索容器样式   z-index: 1000;
    • #toc-pane,#outline-menu-nav  移动端目录面板样式  z-index: 99;
    • .toc-pane-content 页面大纲 z-index: 100;
    • pre .code-fold-btn  代码块底部固定折叠/展开按钮   z-index: 10;
    • pre .code-fold-btn  代码块底部固定折叠/展开按钮   z-index: 10;  （shmaur-js里相同属性也有一条）
    • .code-toolbar-fold  代码块头部任务栏固定折叠/展开按钮   z-index: 1;
  • shmaur-messages
    • .msg  特效字体容器  z-index: 2;    
    • .bubbles  气泡背景   z-index: 1;
  • shmaur-homeCss
    • .g-container .title  标题的伪元素动画   z-index: 1
  • shmaur-navCss
    • #navigationBar  顶部导航栏容器  z-index: 1001;
    • .navbar-links-mobile  移动端全屏导航面板  z-index: 9999;
    • #toggleMenuButton  导航栏汉堡按钮  z-index: 1000;
    • .left-menu-nav  左侧树形目录  z-index: 1;
