  • Akari's -Theme功能简介：
    • 1. 全局配置与 SEO 
        - 读取站点/博主元数据（名称、头像、关键词、备案号等） 
        - 自动生成各页面 SEO Meta、share-Card、robots、canonical 链接 
        - 接入网站管理员设置。让搜索引擎爬取（百度/必应/Google）及 Umami 无 Cookie 站点访问统计
    • 2. 主题样式与脚本资源管理 
        - 动态注入 CSS/JS（highlight、mathjax、viewer、twikoo、折叠、蒙版调色盘、BGM、特效、kimi-chat等） 
        - 按需加载第三方库：css-doodle、force-graph、Sakana! Widget、Twikoo 评论系统、OwO 表情增强
    • 3. 自适应导航栏 
        - PC 端：LOGO、主菜单、下拉子菜单、一键展开“全部菜单”（含最近访问记录）、搜索 
        - 移动端：抽屉式菜单、搜索按钮、菜单切换按钮 
        - 自动高亮当前页面、外链新窗口、最近访问 localStorage 记录与回写
    • 4. 多维度侧边导航（非首页） 
        - 树状/标签/分类 三种维度目录，支持实时搜索、全部展开/折叠 
        - 目录节点自动生成锚点，点击自动路由并定位 
        - 修复移动端首次点击失效、自动滚动高亮当前标题
    • 5. 页面大纲（TOC）与信息卡片 
        - 自动提取 h1-h6 生成层级目录，同步滚动高亮 
        - 信息卡片：博主信息、公告、推荐阅读、文章分类、标签云、最新评论、说说
    • 6. 内容区核心功能 
        - 文章页：自动生成标题、作者、日期、字数、阅读量、标签、版权声明显示 
        - 文章卡片：置顶、摘要、头图、分类、标签、相对时间、外链识别 
        - 自定义页面：友链、朋友圈、现在、归档、标签云+知识图谱可视化 
        - 代码块：行号、语言修正、折叠 
        - 媒体增强：图片点击放大、MP4视频自动替换为可控播放器、数学公式 MathJax 渲染 
        - 打赏模块：微信/支付宝收款码弹窗
    • 7. 交互与体验 
        - 返回顶部、目录/大纲折叠按钮、悬浮提示；打磨细节，优化多个影响体验的小bug，更好的界面排版，更生动的元素。 
        - 智能导航栏（向下滚动隐藏，向上滚动显示，默认注释未启用） 
        - 首页随机语录+气泡+霓虹字特效、彩色飘带背景、蒙版调色盘 
        - 背景音乐
    • 8. 扩展钩子 
        - header 预留插入点，方便不改动模板即可追加 meta、link、script，超详细的中文注释与分组方便二次迭代。
    • 9. 兼容 
        - 打磨移动端细节，移动端优先响应式，兼容主流现代浏览器

「主题根笔记」的全部 标签（label） 和 关系（relation）

它们像「配置项」一样控制静态博客的各种功能，开发新增功需要配置到这里面来。

# ======================= 1. 站点基础信息 ======================== 
# 博客标题、作者、SEO、备案等「一眼能看到的文字」 
siteName                    = 阿卡林の笔记          # 网站大标题（浏览器标签 + 首页） 
bloggerName                 = 阿卡林の笔记          # 作者昵称（版权区 + 侧边栏） 
blogStartTime               = 2025-09-12T00:00      # 建站日期 → 底部「已运行 X 天」 
siteKeywords                = 振动分析、故障诊断...  # SEO keywords 
shareDescription            = 随便记录和总结知识... # SEO description（首页） 
siteDescription             = 使用Trilium Notes...  # <meta name="description"> 
motto                       = 呐!希望您每天都有好心情 # 侧边栏头像下方签名 
record                      = 备案号                # 底部公安备案文字 
recordLink                  = https://beian.miit.gov.cn/ # 备案号跳转链接

# ======================= 2. 视觉 / 风格开关 ===================== 
# 背景图、头像、图标等「长什么样」由它们决定

# ======================= 3. 页面功能总开关 ======================

disableLatestCommentPane    = false                 # 首页「最新评论」侧边栏开关 
enableReward                = true                  # 文章末尾「为 TA 充电」赞赏开关
